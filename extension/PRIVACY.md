# Privacy Policy - GuardianLink

## Data Collection
- URLs scanned are sent to:
  - VirusTotal (https://virustotal.com)
  - AbuseIPDB (https://abuseipdb.com)
  - Google Safe Browsing API
  
- Local logs stored in browser storage only
- No personal data collected
- No data shared except for security analysis

## Data Retention
- Scan logs deleted when extension uninstalled
- Users can manually clear logs via dashboard

## Third-party Services
- VirusTotal: URL reputation analysis
- AbuseIPDB: IP abuse scoring
- Google Safe Browsing: Malware detection
